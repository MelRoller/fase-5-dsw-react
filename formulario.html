1. ¿Cuál es el propósito principal de CSS Grid?

a. Crear diseños bidimensionales (en filas y columnas) de manera más eficiente.
b. Aplicar estilos a elementos HTML.
c. Animar elementos en la página web.

2. ¿Cómo se define un contenedor como grid en CSS?

a. Con la propiedad display: grid;
b. Utilizando la etiqueta <grid>
c. Agregando la clase .grid-container

3. ¿Cuál es la función de la propiedad grid-template-columns en CSS Grid?

a. Establecer el ancho de las filas del grid.
b. Definir el número de columnas que tendrá el grid.
c. Establecer el grosor del borde del grid.

4. ¿Qué elemento HTML se utiliza para organizar los elementos hijos dentro del grid?

a. <grid-item>
b. <grid-element>
c. <div>

5. ¿Cuál de las siguientes propiedades de CSS se utiliza para colocar elementos dentro del grid?

a. grid-column
b. align-items
c. margin
